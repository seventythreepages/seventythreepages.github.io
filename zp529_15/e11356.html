SABRENT CFexpress Type-B カードリーダー USB 3.2 10Gbps対応 (CR-CFER)

速度:USB 3.2 Gen 2は最大10Gbpsのデータ転送速度に対応し、安定した効率的なデータ転送を実現。USB 3.2 Gen 2およびUSB 3.2 Gen 1との下位互換性があり、それぞれの速度制限に対応。
幅広い互換性:最大950MB/秒のCFexpress タイプBカードに対応。USB Type-C OTG対応スマートフォンにも対応。USBの速度制限でThunderboltに対応。
オペレーティングシステム:Mac OS / Windows / Linux / USB On-The-Go (OTG対応)スマートフォンなど。
デザイン:ABSフレームを使用した超薄型アルミケース。洗練され、耐久性があり、便利です。ポータブルでありながら耐久性があり、旅行に最適。
プラグ&プレイ:追加ドライバ不要。バスパワー駆動。外部電源は必要ありません。

Sabrent CFexpress カードリーダーは、ワークフローを加速させる究極のポータブルでコンパクトなソリューションです。 特徴:
■ 速度: USB 3.2は最大10Gbpsのデータ転送速度をサポートし、安定した効率的なデータ転送を実現します。 各速度制限でUSB 3.1とUSB 3.0との下位互換性があります。
■ 幅広い互換性: CFexpress Type-Bカードと互換性があり、最大950MB/sに対応。 USB Type-C OTG対応スマートフォンに対応。 ThunderboltはUSB速度制限に対応しています。
■ デザイン:ABSフレーム付き超薄型アルミケース。 洗練され、耐久性があり、便利です。 ポータブルでありながら耐久性があり、旅行に最適です。
■ プラグアンドプレイ: 追加のドライバーは必要ありません。 バス電源。 外部電源不要。 システム要件:
■ Mac OS / Windows / Linux / USB On-The Go(OTG対応)スマートフォン。 パッケージ内容:
■ CFexpress カードリーダーとライター。
■ USB Type-C - USB Type-Cケーブル。
■ USB Type-CからUSB Type-Aケーブル。
■ 取扱説明書(日本語ではない場合があります)。
